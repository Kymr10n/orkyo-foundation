# Orkyo Reporting Integration API — Implementation Pack

This pack replaces the embedded reporting-engine approach with a tenant-safe Reporting Integration API.

Goal: allow customers to connect external reporting tools such as Power BI, Excel, Metabase, Superset, or custom clients without Orkyo becoming a BI platform.

Files:

1. `01-specification.md` — target architecture and product scope
2. `02-implementation-plan.md` — phased implementation plan for Copilot
3. `03-api-contract.md` — proposed REST endpoints and DTO guidance
4. `04-ui-admin-integration.md` — admin UX and operational controls
5. `05-security-acceptance-tests.md` — tenant isolation and acceptance criteria
